A simple Woocommerce extention for Irish users.

== Description ==
This plugin is a simple plugin for Irish users of Woocommerce. The plugin lists the Counties of Ireland and allows purchases to be made without a postcode making it easy for users. The shipping details are also altered to suit Irish Postal addresses.

Features:

* Counties of Ireland.
* No Postcode required for purchase.
* Tailored shipping forms. 


== Installation ==
1. Upload the "woocommerce-ireland" folder to your "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
